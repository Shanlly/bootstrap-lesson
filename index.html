Glyphicons

History

The Sumerians were the first known civilization to express their ideas in written form. They heavily influenced the later writings of the Egyptian, which we call heiroglyphs. Hieroglyphs and Glyphicons have the same root word glyphē a Greek word meaning carved work, which is from glyphein to carve.

Skip forward some 5200 years and we are using a type of hieroglyphs agian.

Glyphicons are incorporated into the Bootstrap framework. They were originally designed for mobile applications as icons hence glyphICONS for tab bars, toolbars, navigation toolbars or control buttons. As they expand into website and web application they are used as visual navigations.

Customization of Gylphicons

Two easy things you can do are changing the size and color. Below is some examples of doing both of these things

Changing or Defining the Size

Changing or Defining the Color

To increase or decrease the size of your glyphicon, all you need to do is specify in the css or inline in the html itself.

in this example they are defining the size a 6 em.

To change or define the color that you want your glyphicon to be, all you need to do is specify in the css or inline in the html itself.

this changes the glyphicon to blue in both css and html

<html>
	<head>
		<link rel = "stylesheet" href = <link rel = "stylesheet" href = "http://maxcdn.bootstrapcdn.com/bootstrap/3.3.4/css/bootstrap.min.css">
		i.mysize {font-size: 6em;}
	</head>
<body>
	<i class = "glyphicon glyphicon-tree-deciduous mysize"></i>
</body>
</html>
				
<html>
	<head>
		<link rel = "stylesheet" href = <link rel = "stylesheet" href = "http://maxcdn.bootstrapcdn.com/bootstrap/3.3.4/css/bootstrap.min.css">
		i.custom {font-size: 6em; color: blue;}
	</head>
<body>
	<i class = "glyphicon glyphicon-tree-deciduous custom"></i>
</body>
</html>
